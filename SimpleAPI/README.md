# Primeira API com FastAPI

Este projeto foi desenvolvido com o objetivo de compreender os conceitos fundamentais do desenvolvimento de APIs em **Python** utilizando o **FastAPI**. Durante sua criação, explorei a estrutura básica de uma API REST, a definição de rotas, o tratamento de requisições HTTP e a validação de dados com **Pydantic**.

A API utiliza um banco de dados em memória para simular operações de cadastro e consulta de produtos, permitindo praticar os principais métodos HTTP e entender o fluxo de comunicação entre cliente e servidor.

## Funcionalidades

* Listagem de todos os itens cadastrados.
* Busca de um item pelo seu ID.
* Cadastro de novos itens com validação de dados.
* Mensagem de boas-vindas na rota principal.

## Tecnologias utilizadas

* **Python**
* **FastAPI**
* **Pydantic**

> **Objetivo de aprendizado:** compreender os fundamentos do desenvolvimento de APIs REST com FastAPI, praticar a criação de endpoints, a validação de dados e a organização de aplicações backend em Python.
